# app

Next.js **App Router**의 루트 디렉터리입니다. 라우팅·레이아웃·전역 스타일을 담당합니다.

## 역할

- **라우팅**: 파일/폴더 구조가 그대로 URL 경로가 됩니다.
- **레이아웃**: `layout.tsx`로 공통 UI(html, body, 전역 CSS)를 감쌉니다.
- **다국어**: 실제 페이지는 `[locale]` 세그먼트 아래에 있어 `/en`, `/ko` 등으로 접근합니다.

## 구조

| 경로          | 설명                                                               |
| ------------- | ------------------------------------------------------------------ |
| `layout.tsx`  | 루트 레이아웃. `<html>`, `<body>`, `globals.css` 적용.             |
| `globals.css` | 전역 스타일, CSS 변수(테마 색 등), 유틸 클래스.                    |
| `[locale]/`   | 로케일별 라우트. 메인 페이지·레이아웃·상세 페이지가 여기 있습니다. |
| `favicon.ico` | 파비콘.                                                            |

## 참고

- 메인 콘텐츠는 `app/[locale]/page.tsx`에서 렌더됩니다.
- `app/companies/`는 레거시 구조일 수 있으므로, 새 상세 페이지는 `app/[locale]/companies/`를 사용하는 것을 권장합니다.
