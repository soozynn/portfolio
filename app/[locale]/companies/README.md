# app/[locale]/companies

프로젝트 **상세 페이지**용 라우트입니다. (추후 포트폴리오 상세 내용 추가 예정)

## 역할

- 각 프로젝트별 상세 카드/페이지를 `[companyId]` 동적 경로로 제공합니다.
- 예: `/ko/companies/1`, `/en/companies/hyatt-admin` 등.

## 구조

| 경로                   | 설명                                                      |
| ---------------------- | --------------------------------------------------------- |
| `[companyId]/page.tsx` | 동적 라우트. 현재는 placeholder이며, 상세 UI는 추후 구현. |

## 참고

- 메인 페이지의 프로젝트 카드에서 이 경로로 링크하지 않도록 되어 있습니다.
- 상세 내용을 정한 뒤, 카드 클릭 시 여기로 이동하도록 연결하면 됩니다.
