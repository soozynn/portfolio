# components

페이지에서 재사용하는 **UI 컴포넌트** 모음입니다.

## 역할

- 레이아웃·섹션·버튼·메뉴 등 공통 UI를 컴포넌트로 분리합니다.
- `next-intl`의 `useTranslations()`로 다국어 문구를 사용합니다.
- Framer Motion으로 스크롤/등장 애니메이션을 적용합니다.

## 컴포넌트 목록

| 컴포넌트                | 설명                                                                 |
| ----------------------- | -------------------------------------------------------------------- |
| `Header.tsx`            | 상단 네비게이션, 로고, 앵커 링크, 언어 전환, 소셜 링크, 모바일 메뉴. |
| `Hero.tsx`              | 메인 상단 섹션. 이름·한줄소개·이력서 다운로드 버튼·프로필 영역.      |
| `Projects.tsx`          | 프로젝트 섹션. 카드 그리드, 제목·설명 (상세 링크/버튼 없음).         |
| `Technologies.tsx`      | 기술 스택 섹션. 레벨 바 + 추가 기술 및 역량 리스트.                  |
| `AboutMe.tsx`           | 소개/타임라인 섹션. 학력·경력 타임라인.                              |
| `Footer.tsx`            | 하단. Contact 문구, 이메일, 소셜 링크.                               |
| `LocaleSwitcher.tsx`    | 언어 선택 드롭다운 (EN/KO/JA/ZH).                                    |
| `SetDocumentLocale.tsx` | `<html lang="">`를 현재 로케일에 맞게 갱신하는 클라이언트 컴포넌트.  |

## 사용처

- `app/[locale]/page.tsx`에서 위 컴포넌트들을 조합해 한 페이지를 구성합니다.
