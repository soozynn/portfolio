# messages

로케일별 **번역 JSON** 파일이 들어 있는 디렉터리입니다.

## 역할

- `next-intl`이 각 언어별로 사용할 키-값 쌍을 제공합니다.
- 네비게이션, 히어로, 프로젝트, 기술 스택, 소개, 푸터 등 화면에 나오는 문구를 여기서 관리합니다.

## 파일

| 파일 | 로케일 |
|------|--------|
| `en.json` | 영어 |
| `ko.json` | 한국어 |
| `ja.json` | 일본어 |
| `zh.json` | 중국어(간체) |

## 구조 예시

- `nav`: 포트폴리오, 프로젝트, 기술, 소개 등 메뉴 라벨
- `hero`: 인사말, 이름, 역할, 한줄소개, 이력서 다운로드 버튼 문구
- `projects`: 섹션 제목, 각 프로젝트 제목·설명
- `technologies`: 섹션 제목, 기술 항목명, 레벨 라벨, 추가 기술 및 역량 배열
- `about`: 소개 제목, 타임라인(년도·제목·설명)
- `footer`: 연락 문구 (UI에서는 "Contact" 고정)
- `common`, `localeSwitcher`: 공통/언어 선택 문구

컴포넌트에서는 `useTranslations('nav')`, `t('projects.title')`처럼 네임스페이스와 키로 참조합니다.
