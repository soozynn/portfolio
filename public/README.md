# public

**정적 파일**을 두는 디렉터리입니다. 그대로 URL 경로로 제공됩니다.

## 역할

- `/파일명` 형태로 접근 가능합니다. (예: `/resume.pdf` → `public/resume.pdf`)
- 이미지, 아이콘, PDF 등 빌드 없이 그대로 서빙할 때 사용합니다.

## 포함 파일

| 파일                        | 용도                                                                       |
| --------------------------- | -------------------------------------------------------------------------- |
| `resume.pdf`                | 이력서 PDF. (직접 추가 시 Hero의 '이력서 다운로드' 버튼으로 다운로드 가능) |
| `next.svg`, `vercel.svg` 등 | 플레이스홀더/로고용 이미지.                                                |

이 폴더에 넣은 파일은 빌드 결과에 그대로 복사되며, 루트 경로 기준으로 참조하면 됩니다.
